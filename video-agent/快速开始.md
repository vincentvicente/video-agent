### 示例1: 转录本地视频

```bash
cd /Users/vicentezhu/claude-workspace

```

### 示例2: 处理YouTube视频（需要yt-dlp）

```bash
# 1. 安装下载工具（一次性）
brew install yt-dlp

# 2. 修改test-real-video.js中的URL
# 3. 运行
node test-real-video.js
```

### 示例3: 批量处理视频库

```bash
# 创建批处理脚本
cat > batch-process.sh << 'EOF'
#!/bin/bash
for video in data/videos/*.mp4; do
  audio="${video%.mp4}.wav"
  ffmpeg -i "$video" -vn -ar 16000 -ac 1 "$audio"
  ./run-whisper.sh "$audio" "null" "base" "vtt"
done
EOF

chmod +x batch-process.sh
./batch-process.sh
```

## ⚙️ 配置选项

### 选择Whisper模型

在 `.env` 文件中设置：

```bash
WHISPER_MODEL_SIZE=base  # 推荐
```

| 模型 | 速度 | 质量 | 内存 | 推荐场景 |
|------|------|------|------|---------|
| tiny | ⚡️⚡️⚡️⚡️ | ⭐️ | 1GB | 快速测试 |
| base | ⚡️⚡️⚡️ | ⭐️⭐️⭐️ | 1GB | **生产推荐** |
| small | ⚡️⚡️ | ⭐️⭐️⭐️⭐️ | 2GB | 高质量需求 |
| medium | ⚡️ | ⭐️⭐️⭐️⭐️⭐️ | 5GB | 专业转录 |

### 输出格式

```bash
# VTT格式（带时间戳，适合字幕）
./run-whisper.sh "audio.wav" "null" "base" "vtt"

# 纯文本（适合搜索和分析）
./run-whisper.sh "audio.wav" "null" "base" "txt"

# SRT格式（通用字幕格式）
./run-whisper.sh "audio.wav" "null" "base" "srt"

# JSON格式（包含所有元数据）
./run-whisper.sh "audio.wav" "null" "base" "json"
```
