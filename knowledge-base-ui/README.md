# 🎬 视频知识库 Web UI

## ✨ 功能特性

### 📊 视频信息面板
- 视频时长、语言、向量数量
- 转录文本大小统计
- 实时数据展示

### 🔍 语义搜索
- **智能理解**：基于AI的语义搜索，不只是关键词匹配
- **相似度评分**：显示每个结果的匹配度百分比
- **时间戳定位**：精确显示内容所在时间点
- **快速建议**：预设常用搜索词

### 💬 智能问答
- **AI驱动**：使用GPT-4o-mini回答问题
- **上下文理解**：基于视频转录内容回答
- **对话历史**：保留完整对话记录
- **快速提问**：预设常见问题

### 📄 完整转录浏览
- **时间戳对齐**：每句话都有精确时间
- **流畅滚动**：优化的滚动体验
- **大文本支持**：高效渲染长文本
- **即时搜索**：浏览器内Ctrl+F搜索

## 📱 界面说明

### 左上：语义搜索面板
1. 输入搜索词或问题
2. 点击"搜索"或按Enter
3. 查看按相似度排序的结果
4. 点击建议词快速搜索

### 左下：智能问答面板
1. 输入问题
2. 点击"提问"或按Enter
3. AI会基于视频内容回答
4. 对话历史自动保存

### 右侧：完整转录面板
- 滚动浏览完整转录
- 每句话显示时间戳
- 支持浏览器搜索(Ctrl/Cmd+F)

## 🔧 自定义

### 更换视频
编辑 `server.js` 中的 `VIDEO_ID`：
```javascript
const VIDEO_ID = '_VaEjGnHgOI'; // 改为其他视频ID
```

### 更改端口
```javascript
const PORT = 3000; // 改为其他端口
```

### 调整搜索结果数
在 `/api/search` 路由中修改：
```javascript
const results = similarities.slice(0, 5); // 改为其他数量
```

## 🐛 故障排除

### 端口已被占用
```bash
# 查看占用端口的进程
lsof -i:3000

# 停止服务器
./stop-ui.sh
```

### API错误
检查 `.env` 文件中的 `OPENAI_API_KEY` 是否正确

### 知识库未找到
确保已运行过视频处理：
```bash
node process-specific-video.js
```

## ❓ 常见问题

**Q: 可以同时处理多个视频吗？**
A: 当前版本支持单个视频。可以修改代码支持多视频。

**Q: 搜索慢怎么办？**
A: 首次搜索需要连接OpenAI API，后续会更快。

**Q: 可以离线使用吗？**
A: 浏览转录可以离线，搜索和问答需要API。

**Q: 如何添加更多视频？**
A: 运行 `process-specific-video.js` 处理新视频，然后修改VIDEO_ID。
